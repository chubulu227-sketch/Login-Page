<h1>Login Page UI using HTML, CSS, and JavaScript</h1>
<p>This project is a simple and responsive login page created using HTML, CSS, and JavaScript.It includes basic form validation, a clean user interface, and easy-to-understand frontend code suitable for beginners and small web projects.</p>

<h2>Features</h2>
<ul>
<li>Responsive and modern layout</li>
<li>Username & password input fields</li>
<li>JavaScript form validation</li>
<li>Error message handling</li>
<li>Well-structured and clean code</li>
<li>Easy to customize</li>
</ul>
